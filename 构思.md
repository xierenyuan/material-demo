# 项目文件构建

## 目录
__初步构思:__
``` html
  ..  app
    .. _module 业务模块
        ... config 单独模块配置(可写路由 别的一些的module.config 之类的)
        ... controller
            .... module.controller.js
        ... directive
            .... module.directive.js
        ... service
            .... module.service.js
        ... filter
        ... sass 样式文件
            .... index.scss
        ... tmps 模块文件 如果内容少 可直接为html 文件
            .... index.html
        ... module.js  组织当前模块的指令服务控制器的文件  给bootstrap 引用
    .. common 通用模块(同上面的组织)
    .. bootstrap.js  angular 启动文件
    .. entry.js

```
